Mi primer título (h1)

Este es un párrafo. Los párrafos se usan para bloques de texto. Son ideales para articulos, descripciones o historias.

Un subtítulo importante (h2)

Este es otro párrafo. Observa cómo hay un espacio natural entre ellos.
