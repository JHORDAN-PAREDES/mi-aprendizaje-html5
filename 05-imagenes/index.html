Tamaño y proporciones

usa `width` y `height` para controlar el tamaño. Pero ¡CUIDADO! cambiar solo una de estas propiedades puede deformar las imagen.

Buenas Prácticas

  • Siempre usar el atributo "alt": ayuda a personas con discapacidad visual y mejora el SEO.
  • No uses solo "Width" o "Height": Puede distorcionar la imagen.
  • Optimiza tus imágenes: Usa formatos modernos como Webp o png, para maximizar la compatibilidad usar jpeg si es posible.
